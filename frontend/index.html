Technical Summary

Current deployment uses a fine-tuned EfficientNet-B4 dual-head model for hemoglobin regression and four-class anemia severity classification. Inference supports MC-dropout confidence intervals and Grad-CAM explanations.

Model

Primary backbone: EfficientNet-B4, ImageNet-pretrained, fine-tuned on 380×380 RGB images.

Heads: Hb estimate in g/dL plus severity classes: normal, mild, moderate, severe.

Fine-tuning

Two-stage training design: head warm-up, then partial backbone unfreezing for domain adaptation.

Dropout is retained at inference time for uncertainty estimation.

Datasets

Unified dataset pipeline is designed to combine public conjunctiva and nail-bed datasets with planned ICMR field data from Tumakuru.

Current live API has conjunctiva weights loaded; nail-bed deployment remains pending weight upload.

Metrics

Tracked evaluation metrics include MAE, RMSE, Pearson r, AUC, F1, sensitivity, specificity, and Bland-Altman analysis.

Design targets: MAE ≤ 1.0 g/dL, Pearson r ≥ 0.85, AUC ≥ 0.85, sensitivity/specificity ≥ 80%.
